# Overview

利用机器学习方法识别股票图像数据中的关键信息。以Python处理1993-2019年的股票K线数据为基础，并应用卷积神经网络（CNN）实现对股票K线图的预测。通过数据读取与切割、CNN模型搭建与可视化以及模型训练与预测来实现整个流程。

## Requirements

- numpy
- pandas
- tqdm
- torch
- matplotlib
